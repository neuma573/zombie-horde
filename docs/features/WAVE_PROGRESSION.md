# 웨이브 진행 개선

## 상태

- 현재 구현 기준 문서
- 기본 좀비와 기존 무기만 사용
- 러닝 좀비, 신규 무기 및 보상 시스템은 포함하지 않음

## 목적

- 전투와 전투 사이의 흐름을 플레이어가 즉시 이해할 수 있게 한다.
- 웨이브가 진행될수록 적 수와 생성 압박을 완만하게 높인다.
- 게임 판정과 웨이브 전환 시각 효과를 분리한다.
- 큰 delta와 분할 delta에서 웨이브 생성 결과를 보존한다.

## 웨이브 상태

기존 순수 상태 흐름을 유지한다.

```text
waiting
  → spawning
  → active
  → 모든 좀비 제거
  → waiting
```

- `waiting`: 첫 웨이브 준비 또는 웨이브 사이 휴식
- `spawning`: 현재 웨이브의 예정 수를 간격에 따라 생성
- `active`: 예정 수 생성이 끝났으며 살아 있는 좀비 제거를 기다림
- 다음 웨이브는 생성이 끝나고 살아 있는 좀비가 0일 때만 준비한다.

## 난이도 증가

웨이브별 총 좀비 수는 기존 선형 증가를 유지한다.

```text
count = baseZombieCount + (waveNumber - 1) * zombiesPerWave
```

생성 간격은 웨이브마다 20ms씩 감소하되 180ms 아래로 내려가지 않는다.

```text
interval = max(
  minimumSpawnIntervalMs,
  spawnIntervalMs - (waveNumber - 1) * spawnIntervalReductionPerWaveMs
)
```

현재 값:

- 첫 웨이브 준비: 1,000ms
- 웨이브 사이 휴식: 2,000ms
- 최초 생성 간격: 450ms
- 웨이브별 감소: 20ms
- 최소 생성 간격: 180ms
- 첫 웨이브 좀비: 5
- 웨이브별 추가 좀비: 3

체력, 이동 속도, 접촉 피해는 웨이브에 따라 변경하지 않는다.
현재 탄약 보유량은 웨이브 난이도 증가의 상한으로 사용하지 않는다. 무한 탄약 또는
보급 정책은 별도 무기·자원 작업에서 결정한다.

## 표시 정책

상시 HUD:

- 현재 웨이브
- 현재 살아 있는 좀비와 아직 생성되지 않은 좀비를 합한 `LEFT`
- 누적 처치 수

휴식 상태:

- 첫 웨이브 전 `PREPARE`
- 완료한 웨이브 이후 `WAVE N CLEAR`
- 다음 웨이브까지 남은 초 단위 카운트다운

웨이브 시작:

- 웨이브 번호 변경을 감지한 `HudSystem`이 중앙에 `WAVE N`을 짧게 표시한다.
- 이 표시는 Phaser 시각 효과이며 웨이브 상태나 생성 시점을 변경하지 않는다.
- resize 시 현재 viewport의 안전한 중앙 영역으로 재배치한다.
- Scene restart와 shutdown에서는 HUD 수명 주기와 함께 제거한다.

## delta 정책

- `advanceWave`는 전달받은 delta 전체를 소비한다.
- 하나의 delta가 대기와 여러 생성 간격을 지나면 누락 없이 모든 spawn event를 반환한다.
- `GameScene`에서는 렌더 delta가 아니라 기존 1/60초 고정 simulation step으로 호출한다.
- 같은 총 simulation 시간과 같은 생존 좀비 수에는 delta 분할과 무관하게 같은 상태와 생성 수를 반환해야 한다.
- 웨이브 시작 배너의 fade는 시각 효과이므로 렌더 시간으로 처리한다.

## 책임

### `logic/wave.ts`

- 상태 전환
- 웨이브별 좀비 수
- 웨이브별 생성 간격
- delta 소비 및 spawn event 수

### `WaveSystem`

- 순수 웨이브 상태 보관
- `advanceWave` 결과 적용

### `GameScene`

- 살아 있는 좀비 수 전달
- 반환된 수만큼 기존 `SpawnSystem` 호출
- HUD에 현재 웨이브 상태 투영

### `SpawnSystem`

- 기존 상·우·하·좌 순서와 플레이어 최소 생성 거리를 유지
- 좀비 ID 생성

### `HudSystem`

- 준비·클리어 카운트다운 표시
- 웨이브 시작 배너 표시
- 게임 판정에는 관여하지 않음

## 테스트

- 첫 웨이브 준비 시간
- 설정된 수와 간격에 따른 생성
- 살아 있는 좀비가 있을 때 다음 웨이브 차단
- 무한 웨이브 적 수 증가
- 웨이브별 생성 간격 감소와 최소값
- 큰 delta와 분할 delta의 생성 수 및 상태 일치
- 전투 HUD의 남은 적 수
- 첫 준비 및 웨이브 클리어 카운트다운
- portrait와 landscape safe-area 배치
- 기존 생성 순서, 플레이어 최소 거리 및 작은 viewport fallback

## 제외

- 러닝 좀비 및 적 종류별 구성 비율
- 신규 무기
- 보상, 상점 및 업그레이드
- 웨이브별 좀비 체력·속도·피해 증가
- 보스 웨이브
- 오디오
- 스프라이트 및 이미지 에셋
